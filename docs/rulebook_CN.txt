简化战斗

0 属性/调整值

在战斗当中仅考虑属性调整值，下以“调整值”称呼。在战斗开始时列出调整值表以方便战斗。例如：
STR	+2
DEX	+2
END	+2
INT	-1
WIS	-1
CHA	-1
* 所有在DND规则中在战斗内影响属性值的技能，将会被修改为直接影响调整值。

1 回合总骰数

战斗中一回合的总骰数一般为5。通常地，一回合只能进行一个动作。也就是即便选择进行基础掷骰数为1的动作时，亦只能进行一次动作。

1.1 回合先后顺序

回合的行动先后顺序将由动作的基础掷骰数决定。也就是玩家先决定动作再进行结算。当然，玩家也可选择不进行动作，此时的基础掷骰数为0。

1.2 附赠动作和移动

在己方的回合内，除了进行已经决定的动作外，可以将剩下的回合骰数用于移动/附赠动作。附赠动作可以进行任意多个，但是需要消耗对应的回合骰数。用于移动时，每个回合骰数可以移动等同于“移动速度”的距离。

2 基础掷骰数

将攻击掷骰、伤害掷骰、速度统一为基础投掷数。这点如同For the King中的轻攻击和重攻击，在使用武器进行攻击时，玩家可以选择轻攻击，那么该次攻击投掷数较少，优点为快速、易命中，缺点为范围小，伤害较低；也可以选择重攻击，则是牺牲速度和命中率来换取攻击范围与伤害。


2.1 回合顺序与同步回合

在没有其他因素影响的情况下，回合顺序按照投掷数较少的角色先进行。也就是同步回合（先决定行动再进行回合结算），以加快节奏》这点可以讨论？


2.2 攻击的成功骰与失败骰

假设您选择使用巨剑进行一次中速攻击且STR为+0，该次攻击投掷数为3，则掷骰3个六面骰，且将顺序由高到低排列。此时，大于等于4的为“成功骰”，而小于等于3的为“失败骰”。假设您掷出了：

【5】 【3】 【2】

则您掷出了1成功2失败。但若您STR为+2，则从“最大的失败骰”补正+2至“成功骰”的最小阈值。简言之，上述的【5】【3】【2】会变为

【5】 【3+1】【2+1】 = 【5】【4】【3】 = 2成功1失败

若您的STR为负值则会优先扣除“最小的成功骰”。例如，若您的STR为-1且掷出了【6】【4】【1】，则此时计算属性后

【6】【4-1】【1】 = 【6】【3】【1】 = 1成功2失败

2.3 按照失败骰决定命中判定

如果失败骰的数量大于敌方的闪避阈值，则该次攻击被闪避，不进行伤害结算。
从这点开始看出，少骰数的攻击易命中但是伤害低，而多段数的攻击不易命中但伤害高。

2.4 按照成功骰和武器决定伤害判定

对敌方造成的基础伤害为伤害倍率*成功骰，例如倍率1的木剑*2成功骰=2基础伤害。武器、技能提供的特殊效果会影响该伤害的结算；但是为了避免buff叠叠乐造成的找半天，影响“己方伤害结算”的效果被分为（1）伤害倍率加成（2）固定伤害加成。伤害倍率加成为增加伤害倍率，可以理解为武器的锐利度；固定伤害加成则是类似附魔的效果，在伤害倍率*成功骰结算完后，增加固定数值；在某些情况下，附魔也可以造成不同属性的附加伤害（为了结算方便，此类技能一般修改为直接将整个伤害判定改为对应属性（卡蜜尔Q是吧）。

2.5 对方伤害结算

在己方伤害结算完毕后，需要计算来源为敌方的伤害数值调整效果：包括免疫/抵抗/易伤，和护甲。

2.6 免疫/抵抗/易伤

与原版相同。免疫为完全免疫该属性的伤害。抵抗为伤害减少50%（不叠加）。易伤为伤害增加100%（不叠加）。同时具有抵抗和易伤时，则同时失去效果（伤害为原本伤害）。

2.7 护甲

护甲为简单的物理伤害固定减免。例如“己方伤害结算”=20的情况下，打12护甲则伤害变为8。但是，护甲仅对于穿着护甲的部位生效！

2.8 优势/劣势

优势获得的条件大致和原本相同：偷袭、高地等等。当具有优势时，成功骰的阈值-1（不叠加），而劣势时成功骰的阈值+1（不叠加）。当同时具有优势和劣势时则阈值仍为4。

2.9 （笨比）反应/灵巧反应/奥秘

“反应”为触发反应的条件时，可以选择是否立即生效的技能效果。例如“格挡”。没有声明为“灵巧”的反应都是笨比反应——当您触发时笨比反应，会取消当前回合您之后的所有未进行的动作，且之后无法再次触发任何“反应”。

灵巧反应是一种特殊的反应，当触发时不会取消未进行的动作，且之后也可以再次触发。

奥秘类似炉石的奥秘和YGO的陷阱卡，在满足生效条件时会立即生效（您无法选择是否生效）。奥秘不被视为反应，若无特殊声明（奥秘/技能），也不会对该回合后续的动作和反应造成影响

3 控制效果

大部分的控制效果被简化为“禁锢“（无法移动）、“扰乱/沉默”（无法施法，除非使用默发法术）和”失能“（无法移动、动作、反应）。加速以及减速则直接影响每回合的骰数。

4 施展法术/神迹/祷告

法术位系统是否需要更改？（这里模拟法师在施展大规模法术时需要长读条，但效果很爆炸。比如施展某个高环法术时，总共需要投掷15个骰子，但是一个回合只能投掷6个（这只是打个比方，施法速度应该也会受到属性影响）。也就是这个法术总共需要6+6+3，三个回合进行读条，且除非特殊声明，在读条期间是无法移动或反应的。但在结算伤害的时候会计算全部15个骰子。

4.1 专注

为了结算简单，专注法术无法默发。专注不再需要每回合掷骰来进行鉴定，任何附加“沉默”、“失能”的控制会立即打断专注。所有有回合限制的专注法术将持续到战斗结束。
